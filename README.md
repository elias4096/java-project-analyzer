# Getting Started

## Prerequisites

- CMake
- C++ Compiler (at least C++17)

## Installing (Windows 11 + MinGW)

```
git clone --recursive https://github.com/elias4096/java-project-analyzer.git
cd java-project-analyzer

mkdir build
cd build

cmake -G "MinGW Makefiles" ..
mingw32-make

# App
./JavaProjectAnalyzer.exe

# Tests
./Tests.exe
```
